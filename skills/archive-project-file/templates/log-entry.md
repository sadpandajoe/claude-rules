# Development Log Entry Template

Append to the Development Log in PROJECT.md after archiving. One entry per archive operation.

```markdown
### [Timestamp] — Archived: [Phase Name]
- Moved [X] sections to PROJECT_ARCHIVE.md
- Reason: [Why archived]
- PROJECT.md focus now: [Current work]
```

## Rules

- Timestamp in ISO format (`YYYY-MM-DDTHH:MM`).
- Reason is concrete: "phase complete and merged in PR #123" beats "cleanup".
- "Focus now" is one phrase — what the reader should orient on next.
